HTML-effectiveness · pattern 05

Callout / Warning Blocks

Use to lift a note, tip, caveat, or hazard out of the running text. Colour plus an icon sets the severity instantly; one token system keeps all four variants consistent.

ⓘ
Note
Export writes only into the folder you choose. It never touches the live bOS system.
💡
Tip
Re-theme any artifact by overriding the :root tokens — no markup changes needed.
⚠
Warning
Exporting to a non-empty folder overwrites same-named files. Pick a fresh directory.
⛔
Danger
Never paste secrets into a wiki page — exported HTML is plain text and leaves the app.

1

Print the labels

Six ready-made labels. Press Print, then cut them out.

Size 2px
2

Set up the scanner app

In the ScannerToComfortclick app, fill three tabs — then it sends every scan to bOS.

  1. ASettings tabIP · Port · Username · Password (your bOS RPC). Object Path Devices/Scanner, Value Name Value, Min Length 3. → Save Settings.
  2. BScanners tabDetect → scan a label → Add. This whitelists your gun.
  3. CStatus tabTest Connection → Start Listening. Green = ready.
3

Stick the labels

SPOT labels on the three positions. BOX labels on the boxes.

4

Test the scanner

Open any text box and scan one label. It must type the code AND jump to a new line.

✓ Correct
✗ Wrong

No new line? Turn on the Enter / CR suffix in the scanner’s own settings.

5

Scan: spot → box

Scan the position, then the box. Right box = green. Wrong box = red + beep.

✗ Wrong box

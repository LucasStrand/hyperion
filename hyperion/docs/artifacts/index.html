Hyperion · documentation craft

HTML-effectiveness Artifacts

A small gallery of exemplar pages, each demonstrating one pattern that makes documentation effective. They are self-contained, dependency-free, and re-themeable by overriding the :root design tokens — copy a pattern into any wiki page.

01

Progress Dashboard

Labelled bars + percentages for status-at-a-glance across parallel work.

 02

Definition / Term Cards

Glossary units pairing a term, a tight definition, and a worked example.

 03

Step-by-step Playbook

Numbered procedure where each step states intent, action, and a verification.

 04

Comparison Table

Options vs. criteria with a consistent yes / partial / no vocabulary.

 05

Callout / Warning Blocks

Note, tip, warning, and danger admonitions from one token system.

 06

Metric / KPI Tiles

Compact grid of headline numbers, each with a label and a signed delta.

 07

FAQ Accordion

Collapsible Q&A on native <details> — interactive, zero JavaScript.

Each artifact is a standalone HTML document. No external assets, no network, no build step.

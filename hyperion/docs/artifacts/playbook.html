Step-by-step Playbook

Use for procedures the reader will execute. Every step states why, gives the concrete action, and ends with a check so the reader knows it worked before moving on.

  1. Open the project

    Load the workspace so the assistant has a configuration to ground against.

    Run open_project with the project id, or pick it from the launcher.

    The header shows the project name and a node count > 0.

  2. Import the latest .bos snapshot

    Refresh the live system view before asking anything that depends on current state.

    Use import_bos and select the exported snapshot file.

    A diff summary appears and the snapshot timestamp updates.

  3. Export the wiki

    Produce a self-contained handoff folder of the documentation vault.

    Run wiki_export and choose a destination folder.

    The result reports written pages and an index_path.

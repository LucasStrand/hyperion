Definition / Term Cards

Use for glossaries and concept references. The term leads, the definition is one or two sentences, and a worked example grounds it. A coloured rail makes each card a scannable unit.

Node bOS
An addressable element of the bOS object tree — a device, property, or logic block. Every node has a path and a type. e.g. Devices\Modbus\Boiler\Setpoint
Grounding agent
The fenced, untrusted system context spliced into a prompt so the assistant answers about the actual loaded configuration rather than guessing. e.g. the focused node plus its references and children.
Instinct roster
A versioned, per-project block of role guidance that shapes one agent's behaviour. Append-only, so every change is auditable and revertible. e.g. “always cite the node path you read a value from.”

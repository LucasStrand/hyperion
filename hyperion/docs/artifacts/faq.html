FAQ Accordion

Use for question-led reference where most readers want only one answer. Native <details> gives collapse/expand with no JavaScript, so it survives any export.

Does export reach the network?

No. export_site is pure over its inputs and a destination directory. It reads the on-disk wiki and writes static files — no HTTP, no database.

What ends up in the destination folder?

One <slug>.html per wiki page, copied verbatim, plus a generated index.html gallery that links them all.

Can a page slug escape the folder?

No. Slugs are validated against a strict [A-Za-z0-9_-] alphabet, so a page can only ever land directly inside the chosen folder.

How do I re-theme an exported page?

Override the :root custom properties. Every artifact and wiki page reads its colours, radius, and fonts from those tokens.
